#Event Flyer Page - Hans Zimmer Live
A clean, responsive event flyer web page created with semantic HTML5 and custom CSS3. This project showcases a fictional live performance by legendary composer Hans Zimmer at the Sydney Opera House.

##🚀 Features
**Responsive Design**: Built using 90vw widths and Flexbox to ensure the flyer looks great on both mobile and desktop.

**Semantic HTML**: Uses proper tags like <header>, <main>, <section>, and <time> for better SEO and accessibility.

**Modern Aesthetics**:

Dark mode theme with a blended background image.

Hover effects on interactive links.

Custom border-radius and typography scaling.

**Dynamic Layout**: Utilizes flex-flow: row wrap to stack "Music Highlights" and "Special Guests" sections neatly.

##🛠️ Technologies Used
**HTML5**: Structure and content.

**CSS3**: Styling, Flexbox layout, and background blending modes.

Google Fonts/Web Safe Fonts: For clean typography.
